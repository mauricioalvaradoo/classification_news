Internet está lleno de secretos, y uno de ellos es Driving Simulator. Esta función trabaja con  Google Maps  para crear un simulador 3D de manejo y, a pesar del tiempo, se mantiene actualizado. Este simulador de conducción desarrollado por la empresa japonesa Katsuomi Kobayashi, que trabaja para Frame Synthesis, informó la página  El Motor . Sin embargo, para muchos puede parecer desconocido. El sistema de conducción de Google se puede utilizar para distintos objetivos: ya sea para conocer alguna ruta o simplemente con entretenimiento. Además, su uso es gratuito. Para utilizar esta plataforma primero se debe acceder a la web de  Driving Simulator  desde el navegador que se utilice o prefiera. Y una de las facilidades es que no se necesita más accesorios para realizar el manejo. Solo se necesita el teclado. Lo malo es que el simulador solo funciona usando algunas ciudades predeterminadas como Londres, Tokio, Las Vegas, Manhattan, entre otras. Otro punto por destacar es que se puede utilizar desde el móvil o tableta (iOS y Android). También se puede elegir el modo mapa, satelital o ambos mezclados (híbrido). Incluso, Driving Simulator da la opción de conducir también un autobús.
