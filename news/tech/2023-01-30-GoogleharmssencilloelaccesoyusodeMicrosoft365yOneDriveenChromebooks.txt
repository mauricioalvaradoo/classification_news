Google  está trabajando en una nueva integración de Microsoft 365 para los Chromebook, que facilitará la instalación en el sistema operativo  ChromeOS  de la aplicación y la apertura de archivos integrándola directamente en Chromebook Files. Los usuarios pueden  instalar Microsoft 365 y OneDrive a través de una aplicación web progresiva  (PWA, por sus siglas en inglés) para utilizar Microsoft Word, PowerPoint o Excel. Es decir, una aplicación que no depende de ningún sistema operativo, sino que funcionan a través de una página web en el navegador,  tal y como funciona Twitter, por ejemplo. Sin embargo, la compañía ha anunciado  una integración de ambos servicios en Chrome OS . Tal y como ha expresado Google desde la  página de soporte de ChromeOS Beta Tester , es “ importante ” que todos los usuarios de  Chromebook  “ tengan las herramientas disponibles para acceder a los archivos que necesitan para hacer las cosas ”. En este sentido, a finales de este año, los usuarios que dispongan de este paquete de herramientas  podrán instalar la aplicación web de Microsoft 365  y conectar Microsoft OneDrive  con la aplicación Chromebook Files. Con esta mejora, los archivos se moverán automáticamente a Microsoft OneDrive cuando se abran en la aplicación Microsoft 365, según ha explicado Google. Así, estos usuarios, tendrán una “ experiencia de configuración guiada ” que los llevará a través del proceso de  instalación de la aplicación web de Microsoft 365 y OneDrive  a su aplicación Chromebook Files. La compañía ofrecerá más detalles sobre esta nueva integración de Microsoft 365 en los “ próximos meses ”. Sin embargo, estará disponible previamente para aquellos que participen en los canales de desarrollo y beta.
