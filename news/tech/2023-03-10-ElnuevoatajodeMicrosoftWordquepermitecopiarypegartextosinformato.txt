Microsoft  ha anunciado el lanzamiento de una nueva característica que permite pegar texto sin formato con un atajo, que ya está en las versiones beta de Microsoft Word para Windows y Mac. El comando habitual para pegar texto en  Windows  es Ctrl + V mientras que en  Mac  es Cmd + V. Utilizando este sistema, el texto seleccionado se copia con el mismo formato que muestra la versión original. Si se utiliza el ratón para este proceso, al hacer clic con el botón derecho se despliegan opciones de pegado como ‘Mantener formato de origen’, ‘Combinar formato’ y ‘Mantener solo texto’. Microsoft ha introducido una  nueva opción que recibe el nombre de ‘Mantener solo texto’ o ‘Pegar como texto sin formato’ , se corresponde con el atajo de las teclas  Ctrl + Mayús + V para Windows y CMD + Mayús + V en Mac. Para utilizar este atajo, se debe seleccionar un texto o una parte de este de un documento Word, colocar el cursor en la parte del documento en la que se desee copiar el texto y teclear el comando. Entonces,  el texto seleccionado mostrará en su copia el formato original sin tener que utilizar el ratón.
