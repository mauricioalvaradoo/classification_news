Mozilla  ha integrado tres nuevas extensiones para su navegador web Firefox en  Android , que permitirán ocultar la dirección real de correo electrónico, eliminar los elementos de seguimiento en una URL antes de compartirla y escuchar artículos web en vez de leerlos. El navegador web de código abierto dispone de un número limitado de extensiones ‘recomendadas’ que funcionan “como aplicaciones”, tal y como lo describe Mozilla.  Estas extensiones permiten que, entre otras cosas, la navegación sea más rápida y segura. Ahora, Mozilla ha integrado tres nuevas extensiones en esta lista, que ya se pueden descargar a partir de su página web. En primer lugar, ha implementado  Firefox Relay , de desarrollo propio, con la que el usuario podrá ocultar su dirección de correo electrónico real para aumentar su seguridad impidiendo que sea visible públicamente. Firefox Relay utiliza direcciones de correo electrónico proxi, es decir, una dirección de email que puede ser utilizada para dirigir los correos entrantes a otra dirección de ‘email’ principal. De esta forma,  la aplicación genera máscaras de correo electrónico automáticamente con direcciones  de correo aleatorias que terminen en la extensión de Mozilla (@mozmail.com). Otra de las extensiones que ha agregado Mozilla a Firefox es ClearURL . Como su propio nombre indica, se trata de una función que permite limpiar las URL, es decir, la dirección de las páginas web, eliminando los elementos de seguimiento implícitos antes de compartirlas. Los sitios web utilizan elementos de seguimiento para marcar su actividad en línea y que sea rastreable. Sin embargo, tal y como explican desde ClearURL, toda esa información de seguimiento no es necesaria para que un sitio web se muestre o funcione correctamente. Por tanto, lo que plantea esta extensión es  eliminar la parte de la URL  que contiene el seguimiento de forma que el usuario conserve la privacidad al compartir el enlace. Por ejemplo, a la hora de compartir una publicación en redes sociales, se evitaría compartir también estos datos. Finalmente, la tercera extensión que ha incluido Mozilla es  ReadAloud . Con ella,  el usuario podrá escuchar los artículos, ya que es una función que convierte el texto de una página web en audio.  Para ello, solo se ha de activar la función en el menú de Firefox una vez se está dentro del sitio web, que puede ser un blog, una noticia u otras publicaciones. Además, tal y como indican desde ReadAloud, se puede  elegir entre múltiples voces tanto masculinas como femeninas , también se puede configurar el tono de voz y la velocidad de lectura. Esta función es útil para personas que sufren discapacidades de aprendizaje, que están empezando a leer o que simplemente prefieren consumir contenido web mientras hace otras cosas.
